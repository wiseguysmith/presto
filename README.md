# PRESTO Prototype

PRESTO is a polished QR ordering and restaurant operations prototype based on the supplied PRD. It includes a guest ordering flow, live order status, staff kanban, kitchen display, menu manager, QR table workflow, reviews, and analytics with realistic mock data.

Open `index.html` directly in a browser. No package install is required.

## Prototype Notes

- App name: PRESTO
- Mock restaurant: Solana Bistro
- Architecture: static HTML/CSS/JavaScript, organized so the mock data can later move into Next.js, Supabase, Drizzle, and realtime APIs.
- Generated visual asset: `assets/presto-hero.png`
